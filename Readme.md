# About project

Создано с помощью [Create React App](https://github.com/facebook/create-react-app).

[Посмотреть демо](https://gifts-merch.vercel.app/)

## Локально развернуть:
После форка 
```
git clone git@github.com:crucian-afk/gifts-n-merch.git
```
А затем в терминале
```
npm start
```
# На чем написано

* React 17
* Redux, Redux Toolkit
* Typescript
* Ant Design
* Адаптивная верстка
